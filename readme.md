# 🍕 Foodiez – Système de gestion des commandes

Foodiez est une application web de gestion des commandes pour restaurant. Elle permet de créer, suivre, modifier et supprimer des commandes via une interface moderne et dynamique.

Ce projet a été réalisé en équipe dans un cadre éducatif.

## 🚀 Fonctionnalités

### 🏠 Page d’accueil
- Page d’accueil animée et moderne  
- Logo Foodiez stylisé  
- Image de fond  
- Bouton pour accéder au dashboard  

### 📊 Dashboard
- Affichage des statistiques :
  - Total des commandes  
  - Commandes en attente  
  - Commandes acceptées  
  - Commandes terminées  
  - Commandes rejetées  
- Affichage des dernières commandes  

### 📦 Gestion des commandes
- Liste complète des commandes  
- Filtrage par statut (All, Pending, Accepted, Completed, Rejected)  
- Changement de statut d’une commande  
- Suppression d’une commande  

### ➕ Création de commande
- Ajout du nom du client  
- Ajout dynamique des articles (chips input)  
- Saisie du prix total  
- Réinitialisation du formulaire  
- Notifications (toasts)
